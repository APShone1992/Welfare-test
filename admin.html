🚀 Supabase setup (free, 3 minutes)

  1. Go to supabase.com → sign in with GitHub → New project
  2. Wait ~2 min, then open SQL Editor and run: create table sms_log ( id bigint generated always as identity primary key, ts bigint not null, sender_name text default '', phone text default '', type text default '', content text default '' ); create table unresolved ( id bigint generated always as identity primary key, ts bigint not null, query text default '', type text default '' ); create table sessions ( id bigint generated always as identity primary key, ts bigint not null, date text default '', messages int default 0 ); alter table sms_log enable row level security; alter table unresolved enable row level security; alter table sessions enable row level security; create policy "pub read" on sms_log for select using (true); create policy "pub insert" on sms_log for insert with check (true); create policy "pub delete" on sms_log for delete using (true); create policy "pub read" on unresolved for select using (true); create policy "pub insert" on unresolved for insert with check (true); create policy "pub delete" on unresolved for delete using (true); create policy "pub read" on sessions for select using (true); create policy "pub insert" on sessions for insert with check (true);
  3. Go to Settings → API, copy Project URL and anon key
  4. Paste below and click Save & Test
  5. Paste the same into index.html in the Supabase Bridge section
